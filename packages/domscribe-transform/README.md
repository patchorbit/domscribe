# @domscribe/transform

Dev-time source transform that injects stable `data-ds` element IDs and emits entries for the Domscribe manifest. This package provides:

- A small universal transformer for JSX/TSX that injects `data-ds` on host elements (React-first).
- A minimal Vite plugin wrapper to integrate in example apps.

This is Phase M1 of the v1 launch plan. Production stripping and additional framework glue will follow.
